Hygiene Heroes Game Engine

This program allows Hygiene Heroes' designers to easily create a highly customizable educational board games with just a spreadsheet as the configuration file. The board game image, token pieces, and the actions associated with each square on the board can all be specified to relate to the hygiene topic of a game.

Gitlet (Version Control System)

Gitlet is a mini version control system that mimics the functionality of the version control system Git. Gitlet uses SHA1 hashcodes, file persistence, serialization, various data structures, and graph traversals to store and track the contents of files and all of their past versions.
